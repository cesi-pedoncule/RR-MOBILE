# RR Mobile

## Build 

### Needs :  
- Expo account

Run the following command :  

```expo login```

### Run build :  

```eas build -p android --profile preview```  

### Issues to build

- [Install expo SDK 47](https://blog.expo.dev/expo-sdk-47-a0f6f5c038af)

- On failed resolve dependencies :  
```expo doctor --fix-dependencies```
